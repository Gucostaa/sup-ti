## 📝 Resumo do Projeto
O **Sup-TI** é uma solução desenvolvida para centralizar e organizar o fluxo de atendimento técnico empresarial. O projeto surge para substituir métodos manuais (planilhas e e-mails) por uma plataforma robusta que garante rastreabilidade, segurança e agilidade no cumprimento de SLAs.

### 🎯 Objetivos
- Centralizar solicitações de suporte em uma única interface.
- Reduzir o tempo de resposta através de indicadores de prioridade.
- Fornecer dashboards estratégicos para coordenadores de TI.
- Garantir histórico auditável de todas as intervenções técnicas.

---

## 💻 Funcionalidades do MVP (Versão Atual)

### Para Usuários e Técnicos:
- **Dashboard Dark Mode**: Interface intuitiva focada em produtividade.
- **Lista Dinâmica de Chamados**: Organização clara por **Solicitante**, **Setor** e **Assunto**.
- **Gestão de Prioridades**: Identificação visual rápida (Alta, Média e Baixa).
- **Descrição Completa**: Visualização de textos detalhados sem cortes, facilitando o diagnóstico.
- **Filtro em Tempo Real**: Busca instantânea por qualquer campo da tabela.

### Arquitetura do Sistema:
- **API-First**: Backend estruturado em .NET pronto para integrações.
- **Frontend Responsivo**: Desenvolvido em React com Lucide Icons.
- **Persistência**: Banco de dados relacional para histórico completo.
